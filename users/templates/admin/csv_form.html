CSV Format Requirements:

  • Required columns: username, email, role
  • Optional columns: first_name, last_name, parent_username
  • Available roles: SUPER_MANAGER, MANAGER, COLLECTION_OFFICER, CALLING_AGENT
  • parent_username must be an existing user's username
  • File must be in CSV format with UTF-8 encoding
